# 🍞 빵춘쿠키 · 오늘의 빵춘

귀여운 토스터기를 눌러 **그 날 운세에 맞는 빵**을 뽑는 힐링 웹앱입니다.
빵이 부르르 떨리며 튀어나오고, 빵이 건네는 따뜻한 오늘의 운세 한 조각을 받아보세요.

> 동물의 숲 감성 · 카와이 푸드 스타일 · 순수 HTML/CSS/JS (의존성 0)

## ✨ 기능
1. **인트로** — `빵춘쿠키` 로고가 톡 튀어나옵니다. 화면을 누르면 시작.
2. **토스터기** — `오늘의 빵춘` 아래 토스터기(얼굴 없음). 레버를 아래로 드래그하거나 버튼/토스터기를 누르면 1~2초간 부르르 떨리다가 **탁!** 소리와 함께 빵이 튀어나옵니다. (효과음은 Web Audio API, 음원 파일 없음)
3. **결과** — 토스터기는 사라지고 빵 + 운세 문구만 표시됩니다.

## 🥐 빵 5종과 분위기
| 빵 | 테마 | 연출 |
|----|------|------|
| 소금빵 | gloomy | 화면이 우중충해지고 눈물을 뚝뚝 |
| 단팥빵 | smart | 지적인 안경, 차분한 푸른 톤 |
| 식빵 | muscle | 알통 뿜뿜, 근육질 |
| 크루아상 | hip | 선글라스 + 힙한 네온, 결이 다른 무드 |
| 메론빵 | flutter | 두근두근 하트가 둥실 |

## 🎨 색상
- 배경 `#FC9797` · 포인트 `#D1F985`, `#FFE1E7`

## 📁 구성
```
index.html   화면 구조
style.css    스타일 · 테마 · 애니메이션
breads.js    빵 캐릭터(인라인 SVG) + 운세 데이터
script.js    화면 전환 · 레버 드래그 · 사운드 · 랜덤 뽑기
```

## 🚀 로컬 실행
별도 빌드 없이 `index.html`을 브라우저로 열면 됩니다.
(폰트/사운드 정상 동작을 위해 로컬 서버 권장)
```bash
npx serve .        # 또는 아무 정적 서버
```
